# Web Mini Projects (HTML/CSS/JS)

Colección de 3 mini-sitios web hechos con HTML + CSS (y un poco de JavaScript en dos de ellos).
Son proyectos front-end simples, pensados para mostrar estructura, maquetación, estilos y algo de interacción sin frameworks.

## Proyectos incluidos

1) Cafeteria — Café Aurora
Landing page con navegación, sección de menú y diseño limpio.
- Enfoque: maquetación, tipografía, cards y layout responsive básico.
- Carpeta: Cafeteria/

2) Negocio — Andamios Azules
Mini portal para renta de andamios con formulario y lógica en JS.
- Enfoque: formularios, UI de captura y armado de resumen.
- Carpeta: Negocio/

3) Peliculas — MiniFlix
Catálogo tipo streaming con tarjetas y búsqueda.
- Enfoque: grid de cards, estilos “dark UI” y filtrado simple con JS.
- Carpeta: Peliculas/


Dentro de docs/ se encuentran screenshots de los proyectos

Nota: Cada proyecto tiene su propio index.html y recursos locales (CSS/JS/imagenes).

## Cómo ejecutar

Abrir directo
1. Entra a la carpeta del proyecto.
2. Abre index.html con tu navegador.
